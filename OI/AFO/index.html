【AFO】OI 退役记「这是属于我们的容身之地，一场层层叠叠的戏剧」

发表于 2025-10-06 分类于 OI 阅读次数：
这样就退役了吗……

「泠珞，你觉得世界上最令人难过的事情是什么呢？」

「蝴蝶。丑陋的毛毛虫，好不容易可以在梦境中忘记自己的软弱，忘记自己身为害虫的身份，却又生出美丽的翅膀，被光吵醒，被迫意识到自己还活着的事实，然后还要亲手撕开幽居的洞穴。越痛苦，翅膀越充血，它们就越美丽，然后再次为美丽和『醒着就要活下去』的定律而痛苦。它们会飞，却又只能在俗气的花丛中苟且偷生，仰视着永远触及不到的高空的鹰。然后死亡就突然降临，它们连沾沾自喜的这一点高度也被剥夺。它们甚至不能迎来一次壮烈的坠亡，只是轻飘飘地、轻飘飘地……」

「听你这么一说，蝴蝶真的是很可怜啊……」

混沌中，身旁的人儿发出悠长的叹息。

 ——《妄想症 Paranoia》（小说）

PS：本文可能含有少量与 OI 关系不大的内容。无所谓，形散神不散。

PPS：本文含有大量中文 Vocaloid 歌曲歌词，请放心食用。

前言

坐标 HA，但是这好像并不重要了罢。

写到这里的时候，是 2025/10/5，第二年的初赛已经结束了，还在备考今年比赛的同学现在或许在打 AtCoder。

打完 NOIP 的时候，由于打的不太好，心情非常难受，所以开始发疯（见游记），没心思写退役记。之后心思已经在全在文化课了，等后来有时间了就没时间写退役记了（这话怎么怪怪的，不重要）。

当年打 OI 的时候还是很喜欢写各种文章的，比如游记、题解，甚至模拟赛的总结。三年（多一个半月）的 OI 历史，在写最后一篇文章罢。

其实想说的挺多的，毕竟，OI 占据了目前人生的一大部分。

开始吧， g++ heldivis_oi.cpp -o heldivis_oi -O2 ; ./heldivis_oi 。

初识 OI

「如果我只记得起点，却忘记了最初想去的终点的话……」

「……还有没有站在这里的资格呢？」

 ——《初心宇宙》

【初一】

当年初一考完入学考，班主任通知有一个 jzyz 的考试。当天中午在恶补一些奇奇怪怪的奥数题，比如“牛吃草”，听不是很小道的小道消息说还要考“计算机/信息技术”，当时我大概会一些 Python 的输入输出，if、while 等等。但是感觉他要真考代码就太没意思了。

当天下午考完数学，果然还有一门信息学。具体考的什么已经记不清了，只记得有一个经典的上楼梯 Fibonacci 数列问题，数学最后一道题高斯取整没给定义直接写中括号，遂没看懂题；信息学考了二叉树的前中后序遍历——所谓“梦开始的地方”。

然后，2021/10/16，第一次进入 jzyz。当时和文化课同班进入的有 8 位同学，大概是最多的一个班了吧。最开始是上一上午课，先上数学，再上 OI。

学了大概有俩月吧（现在校内 OJ 被封禁了呜呜呜，所以没有具体时间了），是第一次测试。当时数学考的是贼简单的中考内容卷子，但是真不会，喜提 55/100，绝对是人生中数学唯一一次不及格的分数。然后是 OI，是 490/500，rank 1，也是非常的开心啊当时。

初一的这一段上课是挺悠闲的，没有特别大的目的性，纯兴趣。当时的想法是：“把数学当预习蹭蹭课，OI 是什么没听说过”。

值得一提的是，有一次让从北门进学校，然后不知道如何到达机房。于是从高三那边跑了一大圈才找到。没学最短路导致的。

然后在初一下的疫情期间，开始线上授课。反正数学是没怎么听懂，昏昏欲睡。OI……其实也没咋听懂。有一次线上的考试好像，但是考的不是很好。

回来之后，“植树节欢乐赛”，但是因为是学校主动筛人的比赛，并没有那么欢乐。肉眼可见的是人数变少了。原来需要两个机房，现在一个机房人与人之间间隔着坐还能坐下。毕竟 OI 和文化课甚至其他竞赛区别是很大的，非常吃“同频共振”。以后的路，只会越来越难……

然后我爹给我注册了 Luogu 账号，获得了同届同学中（应该是）最小的洛谷 ID：732379。虽然但是当时也没有用就是了。

【初二】

然后到了暑假，有暑期集训。当时还小，啥也不懂，反正我认为集训在独立生活和人际交往方面的帮助大于在 OI 方面的帮助，当然可能是因为普及组确实进度比较慢的原因。印象最深的是 Day12 我们那一片都在打 AtCoder，英文看不懂，翻译更看不懂。最懂的是 -4800 软妹币。

然后就开始 CSP2022J 的准备了。暑假集训后边几天了解到了初赛，做了几套题，还行吧只能说。然后就进入了第一赛季。

第一赛季

少女名为希望　把苦痛都遗忘

眼睛里闪烁着满是希冀的光

 ——《夜妆》

在笔试的训练都是卡线过的情况下，最后 J1 考到了全省的 110 左右，当时也是第一次领略到 CCF 的“成绩未上传，请稍后再查看”，每天很激动，但是回家啥也没有。最先看到的还不是官方成绩。无力吐槽，当然当时也不知道 CCF 是个……好人就没有骂。当然 J1 考的很好也是挺开心的。

然后就出事了。很早宣布由于疫情取消了 J2，本来非常失望以及不满，然后原定的 J2 前一天把小区给封了。哇塞感觉非常庆幸没有错过 J2（我可以错过，但是大家都要错过。这思想是不是不太对）。

然后就学习学习，到了 2 月份有一个省内补测。到时候到了 ZZ，和晚上吃了火锅+冰甜品，晚上就开始闹肚子。于是吹风机被用来艾灸了。

第二天考完感觉很好 355/400 拿下，但是最后挂分挂到 205/400 但是最后还是拿了一等奖，虽然这个一等奖屁用没有。

这年 NOIP 只允许高中生参加（疫情），反正我打的时 J 和 NOIP 无关再说能去也是拉低分数线。这一赛季就结束了。

然后就开始学一些新的东西，比如更难的 DP，字符串等等，第一次听都不是太懂。而且毕竟初二，还不是职业选手，没有那么多时间学 OI。

当时的心态：“那个一等奖方便进 jzyz 强基班，因为不想背政治历史，不想练体育（中考）”。（虽然即使到现在 我们仍不清楚强基班选拔到底有没有参考 OI，我怀疑是被忽悠了。）

然后就是第二轮集训了，共计 -9600。集训前讲了树状数组和线段树，emm，当时不会。然后开始集训，感觉这一次集训效果很好。至少没有像上次一样因为把衣服泡到水盆里跑了好几天导致发出恶臭。虽然最后树上 DP 的那位学长声音小的离谱。然后最后模拟赛被神奇的对数运算给拿下了。（就是问最大值取模后的值，要维护真实值的对数比大小转移才对的一个题。）

当时被 ZZ 的一些人比如算了不点名了压力了，每次模拟赛都是霸榜。据说年龄还比我们小（坏了这不还是点名），压力！然后感觉作为所在的城市被省城的老师阴阳了几句？或许是我太敏感了罢。其实这次集训整体还很好。最大的缺陷是 -4800，建议下个版本优化掉。

然后参加省选。嗯对的没打过 S 去省选，各种部分分尽量了，喜提 6_/600。是的，当年 D1T1 很简单但是我想到了一个线段树做法但是不会线段树于是放弃。

至今 Heldivis 仍不知道 D1T1 的正解如何做，当然其他五道题也不会。

嗯当然了这个省选就是纯纯练习用的。之后该上课上课，周末正常去 jzyz 上 OI 课。

第二赛季

墙角折枝不认命的枝桠　深灰色沙土中书写描画

一座触地却仍洁白的塔　然后赞美它

 ——《象牙塔少女》

【初三】

众所周知打 OI 最难受的地方在于，如果包含 NOI 等等活动，你需要从九月打到下一年七月。关于国赛在七月导致高三生不能打的情况，确实在 OI 没有高考背景的情况下，能让高三牲安安生生地复习，当然关于少一次机会，不做评价。

开始新一赛季，先打 S1，很好。

在 S1 和 S2 中间的一个工作日的夜晚，我在刷手机，看到了 @Rainy7 老师的各式各样的博文。看着她的文章，感觉 OI 真的残酷，毕竟，每年保送的只有 50 人，Ag 也就那么 100 人，但是报 CSP 甚至 NOIP 的竟然有（不想查数据了，毕竟不是报告）人，一将功成万骨枯啊。

然后看着她文章里的一车歌词，入了 VC 的坑。特别是 wxz（包括霾大其他歌）感觉在断章取义的情况下和我当时的心态很符合。额，其实不算断章取义。嗯，所以以后在机房就带耳机听歌，一共搞坏了两个半耳机（第一个是用坏的，线慢慢糟了；第二个是我路过电脑时，一脚把头和线踢断了；第三个是左边声道声音变小了）。

然后打 S2，很不好，135/400，喜提擦边……嗯？竟然不是擦边的一等。有非常经典的一个图：（前情提要：老师问我感觉咋样，我说 T2 35 pts，没想到一遍扫一边计数的方法。）

然后打 NOIP。值得一提的是，因为前一天晚上要提前去 ZZ，所以少上一天晚自习，舒服！

NOIP 挂大分，喜提 181/400，也是过了一等线。感觉还是不错的，但是可爱捏初中生没有奖。

然后就开始回到文化课备战 jzyz 的学科营选拔。当时初一是声称学 OI 对参加 jzyz 学科营有很大帮助，但是出现了这样的情况：我问家长学科营结果咋还没出来。家长让我问问 OI 老师有没有什么消息。OI 老师问家长有什么消息没有，准备报名省选。

虽然学校把我们都蒙在鼓里，总之当然最后还是进了，要不然现在就因为文化课太烂而 AFO 了。

【高零】

初三下学期整个人到了 jzyz。被提前拉走的感觉就是好……吗？第一个晚自习仨小时整个人都是很难受的，感觉还是初中好，有可爱的老师和同学。

大概是省选前一周把下午三四节自习和晚自习占了训练，准备省选。当时有一件很神奇的事情：

文化课老师：你们第三节晚自习（上课前的课间）回来之后（前几分钟）太乱了（在问作业等等）。OIers：好的，我们不回来了。

然后就是省选，Day1 应该是拿了 6_/300，Day2 忘了。当时停听高一大佬讲 D1T1，好吧没听懂。

再说一次，至今 Heldivis 仍不知道 D1T1 的正解如何做，当然其他五道题也不会。

虽然省选还是一坨烂泥，但是我觉得我还是很有希望的嘻嘻。

职业选手

生命的短暂　生活的阻碍　已经逝去的 __ 分之 __

还有多少安排　有多少期待　过去的梦何时捡起来

 ——《一半一半》

当时刚到 jzyz 的时候，OI 老师说我们已经从“过客”变成了“职业选手”，当时没什么感觉，现在才意识到，这就说明高中至少前期生活已经要和 OI 绑定很死了。

于是在很长的一段时间了，周二、周四、周六是从下午四点到十点的 OI 学习，并且周日的下两点到十点都在机房。这一段时间也确实学习到很多知识，熟练度也在提升。

最开始的时候，因为学校还要培养零基础（从初三下被 jzyz 忽悠，啊不是是培养，开始学 OI）的同学，所以学 OI 的时间都是统一的，基本没有耽误文化课（文化课老师有意控制进度和避开培训时间）。但是随着其他同学的退出，就没有这样了。在学习 OI 的时候，和其他同学相比，文化课已经落下了。比如 $y=f(x)$ 的图像关于某一点中心对称，关于 $x=x_0$ 轴对称，求巴拉巴拉，这种题当时完全不会。当然这个现在肯定会了不能像省选那些题一样，毕竟还要高考。当然提起初三下学的高中课程就不得不提起以那位神秘老登，这里不展开了。

当时还是有点慌的，毕竟我感觉我不是能完全考 OI 上学的。听说之后有补课，但是“主要还是需要自己补课”（丁语）。嗯，结果就是高零的时候数学没有 130+ 过，在神秘老登的帮助下的物理也不能稳定上 90，并且考试应该没有进过全班前三。

然后在五月底开始文化课开始准备复习中考的时候，作为 OI 职业选手，我们要去打 ICPC 省赛。当然是练习+团建（？）性质的。

听说上一届学长拿了 1Au + 1Ag，还是有点压力的。不过组了队打去年的省赛模拟，很好 Au 了。然后 ICPC 发生了什么，见游记。这就是所谓深深的无力感吗。当然当时去打 ICPC 其实没啥作用，算是一次比较奇特的经历吧，但是还是有点打击自信心。

打完 ICPC 就回去复习中考了。回到初中校内模拟考据说很好，然后摆了一星期，中考擦边 10 分进 jzyz。其实无所谓，还有分配生。

【高一】

中考结束后五天的假期也被全部改成了 OI 课，然后就回到 jzyz 继续上课，因为已经临近暑假集训，就直接停课了。

这次暑假集训是在提高组。正式集训前照样讲了一些其他东西，当然暑假集训好像还是没用上。（中间有一天上午请了半上午假回学校拿毕业证等等，然后……，下午回去上 OI）集训期间感觉学的还可以，除了电脑是太过流畅了（流畅到整型溢出，所以流畅程度是负数）。打了几次模拟赛，然后发现，省队组竟然有高一的还很有实力（又点名了嘻嘻）。然后算了一下人数，发现进省队难死了。有点难受。但是好在集训的内容学的还行，模拟赛还行。这次，-4800，麻木了。家人们谁懂啊，壹肆肆零零。（PS：后文用“14400”代指为了 OI 花的软妹币，但是显然不止 14400。）

省集训完回家，文化课作业不能写还有学校自己组织的集训，于是直接到了开学。但是不用军训好欸。

开学之后，三星期把暑期集训期间耽误的文化课补了一下。到了 2024/9/21，下午打初赛。打完初赛，就要开始停课了。

第三赛季

取舍

我愿回溯选择　选择与象牙塔的幻影相错　承受注定的偏颇

也不愿在　拥抱过浓墨重彩的热烈后　被迫离开

一些弱智错误

把 luogu 主页的搬过来了。

  • unique() $\to$ 之前要 sort()
  • 多测清空 $\to$ 别用 memset() 用循环（指没有限制 $T$ 限制 $\sum n$ 的题目）
  • $n \neq m \neq k \neq q \neq \dots$
  • 不开 long long 见祖宗
    • #define int long long $\to \text{MLE};;{}_{or};;\text{TLE}$
    • int $\to \text{WA}$
    • typedef long long ll; $\to \text{AC}$
  • 不看 warning 见祖宗：
    • printf("%d\n", ans);
      warning: format ‘%d’ expects argument of type ‘int’, but argument 2 has type ‘long long int’ [-Wformat=]
  • $n\le 10^5\to$ 1ll * n * n
  • 头文件
    • 用万能头 $\to$ 不能用 y1 max
    • 不用万能头 $\to$ 把库写全啊
  • 数据结构
    • 传统线段树 $\to$ seg.Build(n);
    • 分块 $\to$ len = sqrt(n); br[Get(n)] = n;
    • 树剖 $\to$ Dfs2(...) { dfn[x] = ++dfc, ... }
    • 树状数组 $\to$ for (; x < N; x += x & -x) ... for (; x; x -= x & -x) ...
  • 找重儿子（树剖 & DSU on Tree）$\to$ siz[x] = 1;
  • for (auto i : s) 中 auto 是一个大容器的时候要加 &，否则 $\text{TLE}$
  • (res += v) % P $\to\ \text{WA}$
    (res += v) %= P $\to\ \text{AC}$
  • std::cout << tmp << '\n'; $\to\ \text{WA}$
    std::cerr << tmp << '\n'; $\to\ \text{TLE}$
    // std::cerr << tmp << '\n'; $\to\ \text{AC}$
  • char s[N], t[N], n, m; $\to\ \text{RE}$
    char s[N], t[N]; int n, m; $\to\ \text{AC}$

这基本就是一些比较唐的错误。
