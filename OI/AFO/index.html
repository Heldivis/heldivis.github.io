【AFO】OI 退役记「这是属于我们的容身之地，一场层层叠叠的戏剧」

发表于 2025-10-06 分类于 OI 阅读次数：
这样就退役了吗……

「泠珞，你觉得世界上最令人难过的事情是什么呢？」

「蝴蝶。丑陋的毛毛虫，好不容易可以在梦境中忘记自己的软弱，忘记自己身为害虫的身份，却又生出美丽的翅膀，被光吵醒，被迫意识到自己还活着的事实，然后还要亲手撕开幽居的洞穴。越痛苦，翅膀越充血，它们就越美丽，然后再次为美丽和『醒着就要活下去』的定律而痛苦。它们会飞，却又只能在俗气的花丛中苟且偷生，仰视着永远触及不到的高空的鹰。然后死亡就突然降临，它们连沾沾自喜的这一点高度也被剥夺。它们甚至不能迎来一次壮烈的坠亡，只是轻飘飘地、轻飘飘地……」

「听你这么一说，蝴蝶真的是很可怜啊……」

混沌中，身旁的人儿发出悠长的叹息。

 ——《妄想症 Paranoia》（小说）

PS：本文可能含有少量与 OI 关系不大的内容。无所谓，形散神不散。

PPS：本文含有大量中文 Vocaloid 歌曲歌词，请放心食用。

前言

坐标 HA，但是这好像并不重要了罢。

写到这里的时候，是 2025/10/5，第二年的初赛已经结束了，还在备考今年比赛的同学现在或许在打 AtCoder。

当年打完 NOIP 的时候，由于打的不太好，心情非常难受，所以开始发疯（见【NOIP 游记】），没心思写退役记。之后心思已经在全在文化课了，等后来有时间了就没时间写退役记了（这话怎么怪怪的，不重要）。

当年打 OI 的时候还是很喜欢写各种文章的，比如游记、题解，甚至模拟赛的总结。三年（多一个半月）的 OI 历史，在写最后一篇文章罢。

其实想说的挺多的，毕竟，OI 占据了目前人生的一大部分。

开始吧， g++ heldivis_oi.cpp -o heldivis_oi -O2 ; ./heldivis_oi 。

初识 OI

「如果我只记得起点，却忘记了最初想去的终点的话……」

「……还有没有站在这里的资格呢？」

 ——《初心宇宙》

【初一】

当年初一考完入学考，班主任通知有一个 jzyz 的考试。当天中午在恶补一些奇奇怪怪的奥数题，比如“牛吃草”，听不是很小道的小道消息说还要考“计算机/信息技术”，当时我大概会一些 Python 的输入输出，if、while 等等。但是感觉他要真考代码就太没意思了。

当天下午考完数学，果然还有一门信息学。具体考的什么已经记不清了，只记得有一个经典的上楼梯 Fibonacci 数列问题，数学最后一道题高斯取整没给定于只写写中括号没看懂题，信息学考了二叉树的前中后序遍历——所谓“梦开始的地方”。

然后，2021/10/16，第一次进入 jzyz。当时和文化课同班进入的有 8 位同学，大概是最多的一个班了吧。最开始是一上午，先上数学，再上 OI。

学了大概有俩月吧（现在校内 OJ 被封禁了呜呜呜，所以没有具体时间了），是第一次测试。当时数学考的是贼简单的课内卷子，但是真不会，喜提 55/100，绝对是人生中数学唯一一次不及格的分数。然后是 OI，是 490/500，rank 1，也是非常的开心啊当时。

初一的这一段上课是挺悠闲的，没有特别大的目的性，纯兴趣。当时的想法是：“把数学当预习蹭蹭，OI 是什么没听说过”。

值得一提的是，有一次让从北门进学校，然后不知道如何到达机房。于是从高三那边跑了一大圈才找到。没学最短路导致的。

然后在初一下的疫情期间，开始线上授课。反正数学是没怎么听懂，昏昏欲睡。OI……其实也没咋听懂。有一次线上的考试好像，但是考的不是很好。

回来之后，“植树节欢乐赛”，但是因为是学校主动筛人的比赛，并没有那么欢乐。肉眼可见的是人数变少了。原来需要两个机房，现在一个机房人与人之间间隔着坐还能坐下。毕竟 OI 和文化课甚至其他竞赛区别是很大的，非常吃“同频共振”。以后的路，只会越来越难……

然后我爹给我注册了 Luogu 账号，获得了同届同学中（应该是）最小的洛谷 ID：732379。虽然但是当时也没有用就是了。

【初二】

然后到了暑假，有暑期集训。当时还小，啥也不懂，反正我认为集训在独立生活和人际交往方面的帮助大雨在 OI 方面的帮助，当然可能是因为普及组确实进度比较慢的原因。印象最深的是 Day12 我们那一片都在打 AtCoder，英文看不懂，翻译更看不懂。最懂的是 -4800 软妹币。

然后就开始 CSP2022J 的准备了。暑假集训后边几天了解到了初赛，做了几套题，还行吧只能说。然后就进入了第一赛季。

第一赛季

少女名为希望　把苦痛都遗忘

眼睛里闪烁着满是希冀的光

 ——《夜妆》

在笔试的训练都是卡线过的情况下，最后 J1 考到了全省的 110 左右，当时也是第一次领略到 CCF 的“成绩未上传，请稍后再查看”，每天很激动，但是回家啥也没有。最先看到的还不是官方成绩。无力吐槽，当然当时也不知道 CCF 是个……好人就没有骂。当然 J1 考的很好也是挺开心的。

然后就出事了。很早宣布由于疫情取消了 J2，本来非常失望以及不满，然后原定的 J2 前一天把小区给封了。哇塞感觉非常庆幸没有错过 J2（我可以错过，但是大家都要错过。这思想是不是不太对）。

然后就学习学习，到了 2 月份有一个省内补测。到时候到了 ZZ，和晚上吃了火锅+冰甜品，晚上就开始闹肚子。于是吹风机被用来艾灸了。

第二天考完感觉很好 355/400 拿下，但是最后挂分挂到 205/400 但是最后还是拿了一等奖，虽然这个一等奖屁用没有。

这年 NOIP 只允许高中生参加（疫情），反正我打的时 J 和 NOIP 无关再说能去也是拉低分数线。这一赛季就结束了。

然后就开始学一些新的东西，比如更难的 DP，字符串等等，第一次听都不是太懂。而且毕竟初二，还不是职业选手，没有那么多时间学 OI。

当时的心态：“那个一等奖方便进 jzyz 强基班，因为不想背政治历史，不想练体育（中考）”。（虽然即使到现在 我们仍不清楚强基班选拔到底有没有参考 OI，我怀疑是被忽悠了。）

然后就是第二轮集训了，共计 -9600。集训前讲了树状数组和线段树，emm，当时不会。然后开始集训，感觉这一次集训效果很好。至少没有像上次一样因为把衣服泡到水盆里跑了好几天导致发出恶臭。虽然最后树上 DP 的那位学长声音小的离谱。然后最后模拟赛被神奇的对数运算给拿下了。（就是问最大值取模后的值，要维护真实值的对数比大小转移才对的一个题。）

当时被 ZZ 的一些人比如算了不点名了压力了，每次模拟赛都是霸榜。据说年龄还比我们小（坏了这不还是点名），压力！然后感觉作为所在的城市被省城的老师阴阳了几句？或许是我太敏感了罢。其实这次集训整体还很好。最大的缺陷是 -4800，建议下个版本优化掉。

然后参加省选。嗯对的没打过 S 去省选，各种部分分尽量了，喜提 6_/600。是的，当年 D1T1 很简单但是我想到了一个线段树做法但是不会线段树于是放弃。

至今 Heldivis 仍不知道 D1T1 的正解如何做，当然其他五道题也不会。

嗯当然了这个省选就是纯纯练习用的。之后该上课上课，周末正常去 jzyz 上 OI 课。

第二赛季

墙角折枝不认命的枝桠　深灰色沙土中书写描画

一座触地却仍洁白的塔　然后赞美它

 ——《象牙塔少女》

【初三】

众所周知打 OI 最难受的地方在于，如果包含 NOI 等等活动，你需要从九月打到下一年七月。关于国赛在七月导致高三生不能打的情况，当然是在 OI 没有高考背景下给高三复习的安排，当然关于少一次机会，不做评价。

开始新一赛季，先打 S1 （CSP2023 S1 游记），很好。

在 S1 和 S2 中间的一个工作日的夜晚，我在刷手机，看到了 @Rainy7 老师的各式各样的博文。看着她的文章，感觉 OI 真的残酷，毕竟，每年保送的只有 50 人，Ag 也就那么 100 人，但是报 CSP 甚至 NOIP 的竟然有（不想查数据了，毕竟不是报告）人，一将功成万骨枯啊。

然后看着她文章里的一车歌词，入了 VC 的坑。特别是 wxz（包括霾大其他歌）感觉在断章取义的情况下和我当时的心态很符合。嗯，所以以后在机房就带耳机听歌，一共搞坏了两个半耳机（第一个是用坏的，线慢慢坏了；第二个是路过时，一脚把头和线踢断了；第三个是左边声音变小了）。

然后打 S2（CSP2023 S2 游记），很不好，135/400，喜提擦边……嗯？竟然不是擦边的一等。有非常经典的一个图：（前情提要：老师问我感觉咋样，我说 T2 35 pts，没想到一遍扫一边计数的方法。）

然后打 NOIP（NOIP2023 游记）。值得一提的是，因为前一天晚上要提前去 ZZ，所以少上一天晚自习，舒服！

NOIP 挂大分，喜提 181/400，也是过了一等线。感觉还是不错的，但是可爱捏初中生没有奖。

然后就开始回到文化课备战 jzyz 的学科营选拔。当时初一是声称学 OI 对参加 jzyz 学科营有很大帮助，但是出现了这样的情况：我问家长学科营结果咋还没出来。家长让我问问 OI 老师有没有什么消息。OI 老师问家长有什么消息没有，准备报名省选。

虽然学校把我们都蒙在鼓里，当然不排除……的可能性，总之当然最后还是进了，要不然现在就因为文化课备战而 AFO 了。

【高零】

初三下学期整个人到了 jzyz。被提前拉走的感觉就是好。然后第一个晚自习仨小时整个人都是很难受的，感觉还是初中好，有可爱的老师和同学。

大概是省选前一周把下午三四节自习和晚自习占了训练，准备省选。当时有一件很神奇的事情：

文化课老师：你们第三节晚自习（上课前的课间）回来之后（前几分钟）太乱了（在问作业等等）。OIers：好的，我们不回来了。

然后就是省选，Day1 应该是拿了 6_/300，Day2 忘了。当时停听高一大佬讲 D1T1，好吧没听懂。

再说一次，至今 Heldivis 仍不知道 D1T1 的正解如何做，当然其他五道题也不会。

虽然省选还是一坨烂泥，但是我觉得我还是很有希望的嘻嘻。

职业选手

生命的短暂　生活的阻碍　已经逝去的 __ 分之 __

还有多少安排　有多少期待　过去的梦何时捡起来

 ——《一半一半》

当时刚到 jzyz 的时候，OI 老师说我们已经从“过客”变成了“职业选手”，当时没什么感觉，现在才意识到，这就说明高中至少前期生活已经要和 OI 绑定很死了。

于是在很长的一段时间了，周二、周四、周六是从下午四点到十点的 OI 学习，并且周日的下两点到十点都在机房。这一段时间也确实学习到很多知识，熟练度也在提升。

最开始的时候，因为学校还要培养零基础（从初三下开始学 OI）的同学，所以学 OI 的时间都是统一的，基本没有耽误文化课。但是随着其他同学的退出，在学习 OI 的时候，其他同学学的文化课已经落下了。比如 $y=f(x)$ 的图像关于某一点中心对称，关于 $x=x_0$ 轴对称，求巴拉巴拉。当然这个现在肯定会了不能想省选那些题一样。当然提起初三下学的高中课程就不得不提起以那位神秘老登。

当时还是有点慌的，毕竟我感觉我不是能完全考 OI 上学的。听说之后有补课，但是“主要还是需要自己补课”（丁语）。嗯，结果就是高零的时候数学没有 130+ 过，在神秘老登的帮助下的物理也不能稳定上 90，并且考试（应该要除了中考二模）没有进过全班前三（因为跨班比较不好描述）。

然后在五月底开始文化课开始准备复习中考的时候，作为 OI 职业选手，我们要去打 ICPC 省赛。当然是练习+团建（？）性质的。

听说上一届学长拿了 1Au + 1Ag，还是有点压力的。不过组了队打去年的省赛模拟，很好 Au 了。然后 ICPC 发生了什么，见 HAICPC2024 游记。这就是所谓深深的无力感吗。当然当时去打 ICPC 其实没啥作业，算是一次比较奇特的经历吧，但是还是有点打击自信心。

打完 ICPC 就回去复习中考了。回到初中校内模拟考据说很好，然后摆了一星期，中考擦边 10 分进 jzyz。其实无所谓，还有分配生。可见……还是不太好，不利于……，非常恶心人。

【高一】

中考结束后五天的假期也被全部改成了 OI 课，然后就回到 jzyz 继续上课，因为已经临近暑假集训，就直接停课了。

这次暑假集训（HASC2024 游记）是在提高组。正式集训前照样讲了一些其他东西，当然暑假集训好像还是没用上。（中间有一天上午请了半上午假回学校拿毕业证等等，然后……，下午回去上 OI）集训期间感觉学的还可以，除了电脑是太过流畅了（流畅到整型溢出，所以流畅程度是负数）。打了几次模拟赛，然后发现，省队组竟然有高一的还很有实力（又点名了嘻嘻）。然后算了一下人数，发现进省队难死了。有点难受。但是好在集训的内容学的还行，模拟赛还行。这次，-4800，麻木了。家人们谁懂啊，壹肆肆零零。（PS：后文用“14400”代指为了 OI 花的软妹币，显然不止 14400。）

省集训完回家，……，文化课作业不能写还要有学校的集训，于是直接到了开学。欸不用军训好欸。

开学之后，三星期把暑期集训期间耽误的文化课补了一下。到了 2024/9/21，下午打初赛。打完初赛，就要开始停课了。

第三赛季

取舍

我愿回溯选择　选择与象牙塔的幻影相错　承受注定的偏颇

也不愿在　拥抱过浓墨重彩的热烈后　被迫离开

 ——《八重回归》

显然，能开始考虑是否停课的 OIer 都是有一定实力的，换言之，都是有脑子的。所以对于是否停课，都是要考虑一下的。

竞赛和高考之间的关系是一个经久不衰的话题……大部分竞赛生都有过或长或短的停课经历……长时间停课必然严重影响文化课成绩，这也成了家长和老师对竞赛学习最大的顾虑……其一……如果能够依靠自己的竞赛成绩取得各大高校的降一本乃至保送资格，就可以摆脱绝大部分的升学压力……其二，大部分竞赛生都有一个同样的看法，即文化课和竞赛学习相比起来非常无聊……但实际上，对家长和校方而言，是否同意竞赛生停课本质上是一个性价比问题，即让选手们将一部分学习文化课的时间转移到竞赛学习上能否获得更好的升学优势……

 ——《整型溢出：信息学竞赛的发展，繁荣与衰退》

所以，问题就在于“性价比”——停课到底值不值。

如果停课，讨论一下：如果没有进省队的希望，那么两个月换一等值不值？如果有进省队的希望，再停课几个月，能不能进省队？如果能进省队，再停课，这样就一年了。国赛能拿奖吗？…………总之，只要停课，就只要两个选项——不值得，或者无底洞。

但是我觉得还是要冲一下，也决定（只要不是 NOIP 考的全省第一，显然不可能）NOIP 就是最后一战。

所以为啥要停课，用个不太恰当的词就是“恋爱脑”。嗯，还是很恰当的。

唉，人有时候是很冲动或者固执的，一旦内心有了决定，甚至是源于偶然起意或有其他私心，那么不管怎么考虑，再怎么客观看待优劣，都是会主观地通过给这些影响加权，从而将内心已有的抉择看待为一项更优的选择作为结束。

CSPS2024，启动！

S1（CSP2024 S1 游记）还是很好的。考试时看到监考的 QY2002 笑嘻嘻，看到巡视的 wxq 不嘻嘻。考完回到机房开始刷复赛题，以及 AtCoder。泄题就泄题吧，无所谓。

S2（CSP2024 S2 游记）考试感觉状态不错，除了 T2 浪费时间太长以及 T3 不会正解以及 T4 没写完暴力，挂了点分，但是成绩还算喜人，感觉按照这个排名 NOIP 一等稳了。顺手拿了个洛谷钩子。唉，有时候，算了不感慨了，你谷钩子有的蛋用。

之后，学习了：根号分治长链剖分线段树合并直径和基环树摩尔投票ExGCDCRTExCRT莫二离欧拉回路DSUOnTree图论插入DP动态DP贼难的DP2SAT线段树优化建图Boruvka同余最短路的后半部分。（不想找日期了）（感谢某位在役同学提供的账号）

然后自己刷了刷题，打了打板子，就去 NOIP 了。

唉，往事不堪回首（NOIP2024 游记），这次真的是最后一舞了。（PS：NOIP 一等了。）

彩虹糖（见游记）是不够吃的，就像我的 OI 一样，是没有结局的。

三年 OI 一场空，开了 long long 照样见祖宗。

打完 NOIP 逛了一圈 ZZU，以后不会再来了。

回到车上，……，到家之后，就要准备第二天回班上文化课了。

哦对了，（本人物化生）政治必修二（同学说还没讲完）丢机房了，然后又买了一本，过两天（其他人）在机房找到了，然后老师不讲了。-11，和 14400 一样心疼。

其实没有 14400 心疼。

现实

弱蝶挣扎着逃出了蛹　载着苍白的梦想迎接坠落

只为在希望消失的时候　坚定说 再狼狈也是活过

请抬起头　看深渊上空　现实旁观宛如残酷孩童

推搡着你成为受害者一个　从未留心应允任何因果

至高的殿堂浑然不觉　只留我们自顾自抗争

再一次向现实宣战后　从头来过

再一次向未来朝圣后　从头来过

 ——《妄想 Reality》

怎么说呢，打了三年的 OI，学了这么久计算机相关，一下子从 OIer 飞升为 WHKer，还是有一定落差的。

于是开始补课。补完了也高一上期末了。文化课成绩从高一上开学考班级 rank 3 变成了高一上期末考班级 rank 16。去年 @forgive_ 学长回来直接怼到市一，%%%。不过没关系，过了个暑假，高一下回来 rank 10 2 2 2 2 10（不是能不能让我期末考好一次！）。

同时后数学老师给我拉走学 MO 了，但是 MO 和 OI 的差别还是很大的，OI 的思维方式已经固定了，接受不了 MO 思维，遂退。
